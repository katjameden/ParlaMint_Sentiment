# ParlaMint_Sentiment: Workflow for sentiment annotation and aggregation of parliamentary debates

## Sentiment annotation on the utterance level

This part of the repository contains settings for manual sentiment annotation at the utterance level and data analysis.

## Directories

**[Annotation_Main](./Annotation_Main):** Samples and results for the main annotation

**[Annotations_Tests](./Annotations_Tests)**: Test runs to optimise the annotation workflow (and their results)

**[Datasets](./Datasets)**: Annotation samples (test and main)

**[Notebooks](./Notebooks)**: Jupyter notebooks and other scripts used for the conversions and development
