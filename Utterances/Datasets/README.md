README

The folder contains datasets for the sentiment annotation/aggregation campaing to annotate parliamentary debates on the level of utterance (ParlaMint-SI).


Aggregation datasets: 

- Sample.csv: Sample of the full 1000 utterances to be annotated within the campaign
- Sample_dev.csv: Development split of the original sample (Sample.csv)
- Sample_test.csv: Test split of the original sample (ParlaMint_sample.csv) 
<!--Not to be touched until the test of the aggregation methods-->

Annotation test datasets: 

- Test1_sample.csv: Sample of only 19 utterances for test round 1
- Test2_sample.csv: Sample of 50 utterances for test round 2
- Test3_sample.csv: Sample of 50 utterances for the test round 3

Main annotation datasets: 
(Excel versions of the samples include the stylized version of the sample (hidden columns, drop-down values) 
used for the actual annotations.  

- Main1_sample.csv: Sample of 200 utterances, first main annotation round
- Main2_sample.csv: Sample of 200 utterances, second main annotation round
- Main3_sample.csv: Sample of 200 utterances, third main annotation round
- Main4_sample.csv: Sample of 200 utterances, 4th main annotation round
- Main5_sample.csv: Sample of 200 utterances, last main annotation round
