The folder contains datasets for the sentiment annotation/aggregation campaign to annotate parliamentary debates on the level of utterance (ParlaMint-SI).

- `Annotations.csv`: The dataset with utterance-level annotations, annotator comments and the consolidated final labels. 
Aggregation datasets:

- `Sample.csv`: Sample of 1,000 utterances selected for annotation during the main annotation campaign. 
- `Sample_dev.csv`: Development split derived from the original sample (`Sample.csv`).
- `Sample_test.csv`: Test split derived from the original sample (`Sample.csv`). **Do not modify this file until the aggregation methods have been evaluated.**

Annotation test datasets:

- `Test1_sample.csv`: Sample of only 19 utterances for test round 1
- `Test2_sample.csv`: Sample of 50 utterances for test round 2
- `Test3_sample.csv`: Sample of 50 utterances for the test round 3

Main annotation datasets: (Excel versions of the samples include the stylized version of the sample, e.g. hidden columns, drop-down values) used for the actual annotations.

- `Main1_sample.csv`: Sample of 200 utterances, first main annotation round 
- `Main2_sample.csv`: Sample of 200 utterances, second main annotation round 
- `Main3_sample.csv`: Sample of 200 utterances, third main annotation round 
- `Main4_sample.csv`: Sample of 200 utterances, fourth main annotation round 
- `Main5_sample.csv`: Sample of 200 utterances, last main annotation round
