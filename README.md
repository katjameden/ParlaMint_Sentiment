# ParlaMint_Sentiment: Workflow for sentiment annotation and aggregation of parliamentary debates

The repository contains settings for manual sentiment annotation at the utterance level and data analysis.
In addition, the workflow includes experiments to aggregate sentence-level scores, created during the previous ParlaSent annotation campaign ([ParlaSent dataset](http://hdl.handle.net/11356/1868)) to produce utterance-level scores.
The testing methods will range from simplistic approaches (such as average or mean) to more complex techniques aiming to identify the most suitable method for aggregation.

## Directories
**[Datasets](/Datasets)**: Annotation samples (test and main)

**[Annotation_Tests](/Annotation_Tests)**: Test runs to optimise the annotation workflow (and their results)

**[Annotation_Main](/Annotation_Main):** Samples and results for the main annotation

**[Notebooks](/Notebooks)**: Jupyter notebooks and other scripts used for the conversions and development 

**Aggregation**: Tests and development of the aggregation methods
